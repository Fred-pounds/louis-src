# SRC Week 2026 | St. Louis Senior High School

## Project Info

**Description**: For Students, For Truth, For Progress. Join us for a week of activities, workshops, sports, and celebrations at St. Louis Senior High School.

## Getting Started

### Prerequisites

- Node.js & npm installed

### Installation

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd src-week-hub

# Step 3: Install dependencies
npm install

# Step 4: Start the development server
npm run dev
```

## Technologies Used

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Deployment

Build the project for production:

```sh
npm run build
```
